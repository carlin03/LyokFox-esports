# LyokFox Esports — Guia rapida

## Mismas paginas en TODAS las webs (igual que localhost)

Localhost y online usan los MISMOS archivos y rutas .html:

| Pagina     | Ruta              |
|------------|-------------------|
| Inicio     | index.html        |
| Comunidad  | comunidad.html    |
| Noticias   | noticias.html     |
| Historia   | historia.html     |
| Equipos    | equipos.html      |
| Mi cuenta  | cuenta.html       |
| Sponsor    | sponsor.html      |
| Contacto   | contactanos.html  |

Enlaces internos: siempre relativos (ej. comunidad.html#tienda).
Config central: js/config.js
